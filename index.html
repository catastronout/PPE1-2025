Miniprojet PPE1

Le but de ce miniprojet est de récupérer une liste d'URLs, de télécharger les pages correspondantes, puis d'extraire les informations suivantes :

  • le code HTTP,
  • l'encorage,
  • le nombre de mots contenus dans la page.
